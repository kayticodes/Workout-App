# Workout-App
This is a full-stack web application (SPA) that tracks workout routines entered by the user.

## Technologies
- HTML
- CSS
- JavaScript
- MongoDB
- Express.js
- React
- Node.js

## Gif Walkthrough 
### Front-end
The React app includes the following 3 pages:
- Home Page
- Edit Workout Page
- Create Workout Page
